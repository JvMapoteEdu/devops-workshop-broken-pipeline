# 🚀 DevOps Workshop – Broken CI/CD Pipeline Challenge

Welcome to the DevOps hands-on activity!

## 🎯 Objective

Fix the pipeline so that it completes successfully ✅

---

## 🧠 What You Will Learn

- What CI/CD is
- How pipelines work
- How to read logs in GitHub Actions
- How to debug pipeline errors
